Data Science

Course Title: Introduction Data Science (4-Week Course)

Week 1: Introduction to Data Science

What is Data Science?

-The data science process

-Key roles in data science teams

-Overview of common data science tools and languages (Python, R)

Week 2: Data Acquisition and Cleaning

-Data sources and formats (structured, unstructured)

-Data collection and scraping

-Data preprocessing and cleaning techniques

Week 3: Exploratory Data Analysis (EDA)

-Descriptive statistics

-Data visualization using Python libraries (e.g., Matplotlib, Seaborn)

-Identifying patterns and outliers

Week 4:Introduction to Machine Learning

-What is machine learning?

-Types of machine learning (supervised, unsupervised, reinforcement learning)

-Hands-on example: Building a simple machine learning model using scikit-learn

Assessment: - Weekly quizzes or assignments

- Final project: Conduct EDA on a provided dataset and build a basic machine learning model

- Class participation and discussions
